# sav2csv.py.script
Script que traduce un archivo SAV (base de datos estadísticos) a CSV (texto separado por comas)

## Archivos SAV

Son archivos de datos generados a partir del software de análisis estadístico IBM SPSS. Este software requiere licenciamiento pago, y no es muy ecónomico, por esto es que se busca una alternativa para acceder a los datos generados a partir del mismo.

Se puede utilizar este script como forma ágil para acceder a los datos por ejemplo desde Microsoft Excel u otra herramienta de la familia de las hojas de cálculo ya que todas permiten importar a partir de CSV.

Opcionalmente, se podría utilizar el software GNU PSPP de licenciamiento libre y que también esta diseñado para análisis estadístico.
