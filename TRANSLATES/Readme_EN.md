# sav2csv.py.script

[Traducir al Español](Readme.md)

Script that translates a SAV file (statistical database) to CSV (comma-separated values)

## SAV Files

These are data files generated by the IBM SPSS statistical analysis software. This software requires a paid license and is not very economical, so an alternative way to access the generated data is being sought.

This script can be used as a quick way to access the data, for example, from Microsoft Excel or other spreadsheet programs, since they all allow importing from CSV files.

Optionally, the free and open-source GNU PSPP software, also designed for statistical analysis, could be used.
